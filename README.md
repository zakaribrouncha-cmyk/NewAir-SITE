# NewAir-SITE

Site vitrine professionnel pour NewAir RP.

## Pages ajoutées

- `index.html` : accueil avec logo animé au centre
- `map.html` : carte interactive avec points importants
- `whitelist.html` : interface whitelist en 4 étapes
- `login.html` : page connexion au style Discord
- `compte.html` : espace joueur
- `reglement.html` : règlement
- `equipe.html` : page équipe/staff

## Discord

Le bouton Discord utilise l'invitation : `https://discord.gg/UyzapQ2zap`.

Pour faire une vraie liaison de compte sécurisée, il faudra ajouter un backend ou un bot Discord côté serveur. Le fichier `assets/auth-config.js` est prêt pour mettre ton Client ID Discord.
